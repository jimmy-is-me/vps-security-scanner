📖 專案說明
VPS 安全掃描與清除工具 v4.4.1 是一款專為Linux VPS設計的輕量級安全檢測工具,採用完全無痕跡設計理念,不安裝任何第三方掃描軟體,僅使用系統內建指令進行安全檢測,掃描完成後可選擇自動刪除腳本本身,不留下任何痕跡。

🎯 設計理念
傳統資安掃描工具如Maldet、ClamAV或Nessus雖然功能強大,但往往需要佔用大量系統資源(4-8GB RAM、30-50% CPU),且會在系統中留下安裝檔案與日誌記錄。本工具針對小型VPS環境優化,使用不到100MB記憶體、CPU使用率低於5%,掃描時間僅需30-60秒,適合管理1-200個網站的主機環境。

✨ 核心特色
🚀 完全無痕跡 - 不安裝工具、不寫日誌、可自動刪除腳本

⚡ 高效能設計 - 使用xargs平行處理與陣列避免子shell,優化CPU使用

🛡️ 自動防護 - 整合Fail2Ban自動安裝,48小時IP封鎖機制

🎨 視覺化介面 - 彩色輸出、進度提示、風險等級標示

🔍 12項全面檢測 - 涵蓋惡意程式、Webshell、挖礦程式、暴力破解等威脅

🔧 適用場景
WordPress網站管理者需要快速檢查主機安全

VPS運維人員進行定期安全巡檢

發現異常流量或資源佔用時的緊急排查

網站遷移前後的安全驗證

多租戶主機環境的例行掃描(支援XCloud、FlyWP、CloudPanel等面板)

本工具已針對v4.4.1版本進行CPU使用優化、Webshell掃描加速(移除時間限制並限制20筆結果),以及Fail2Ban封鎖IP詳細資訊顯示(含最後嘗試時間與剩餘封鎖時長)。
